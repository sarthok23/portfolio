# Portfolio Website - Md. Muhaiminul Islam

A modern, responsive portfolio website showcasing professional experience, projects, and skills.

## 🚀 Features

- **Modern Design**: Glassmorphism effects, particle animations, and smooth transitions
- **Dark/Light Mode**: Toggle between themes with persistent settings
- **Fully Responsive**: Optimized for all devices and screen sizes
- **Interactive Animations**: Counter animations, parallax effects, and scroll reveals
- **Professional Sections**: Hero, About, Experience, Projects, Skills, Awards, Activities, Contact

## 📸 Adding Your Profile Photo

To add your profile photo:

1. Save your profile picture as `profile.jpg` in the same folder as `index.html`
2. Make sure the image is:
   - **Portrait orientation** (recommended 3:4 ratio, like 300x400px)
   - High quality (at least 400x500px for best results)
   - Professional headshot or full upper body
   - In JPG, PNG, or WebP format

The website will display your full photo in a modern frame with animated background effects and professional styling. The photo will be automatically cropped to fit the portrait frame beautifully.

## 🛠 Customization

### Personal Information
- Update contact details in the HTML file
- Modify experience, education, and project information
- Add or remove skills and technologies

### Styling
- Colors and themes can be modified in `styles.css`
- Animation timing and effects in `script.js`
